Project 1: Documentation

When I started working on project 1, I started by taking a look at the material provided by the teacher. I went on the teachers website. From there I hovered over Assignments and pressed 1. Documentation. What I found particularly helpful was the additional resources for html preperations. I started by going over the exercises on w3schools. They taught me the basics of html5 programming which came in really handy when I wanted to change templates such as the one I am using right now. In the beginning I wanted to create my own website from scratch. I started by creating a sidebar menu with this tutorial. However, after two hours of creating the sidebar menu I wanted to create a design for the website. However, I kind of lost my motivation to create this website from scratch because it would take so much effort. That is when I decided to use a free template. I found the template I am using for this website on HTML5UP which is a website that offers a lot of free html5 templates. This template is called editorial and can be found here. I used Brackets to edit my code and I connected Brackets to Github with this Tutorial. I start by cloning the repository from Github to my existing project in Brackets. That way I can make changes locally on Brackets and then commit and push my changes to the remote repository on Github can be downloaded here.
